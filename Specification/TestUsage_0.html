    <colspec>
    Specifies a column, a vertical portion of a <table>.
    <empty/>
    @colnum

    The value of colnum is not useful to identify a column in an <entry>, so serves no functional purpose other than a consistency check on the order of the <colspec>s.

    Number of column, counting from 1 at left of the table.
    @colname
    Name of column, used to specify the position or horizontal span of columns in a row by reference in <entry> using colname, namest, and/or nameend.
    @colwidth
    Either proportional measure of the form number* or fixed measure (pt, pi, pica).
    @colsep
    Default for column ruling to the right of <entry>s starting in this column (within the <tgroup>).
    @rowsep
    Default for row ruling below an <entry> starting in this column when there is neither a rowsep value on the <entry> nor on the <row>.
    @align
